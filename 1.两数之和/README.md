# 两数之和

## 题目描述

给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个 整数，并返回他们的数组下标。

你可以假设每种输入只会对应一个答案。但是，你不能重复利用这个数组中同样的元素。

示例:

>给定 nums = [2, 7, 11, 15], target = 9
>因为 nums[0] + nums[1] = 2 + 7 = 9
>所以返回 [0, 1]

## 笔记

* 这个题目很容易想到暴力解法，难点在于性能优化
* 我使用了STL库的map，一种用红黑树存储的数据结构。把整数作为key，下标作为value
* 问题：相同数字没法insert进去
* 解决：使用double类型存储，用小数位区分同一个整数（不用float因为大整*数float位数不够会出错QAQ）
* 233：multimap可以存相同key
* 另外因为题目假设只输出一个答案，所以可以尝试在建map表的查找是否有两个数满足题意，在数据量小的时候可以明显提高效率。
